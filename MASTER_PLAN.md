# [MASTER PLAN] 모바일 연동 하이브리드 CRM "비즈커넥트"

**문서 버전:** 1.0 (Final Draft)  
**작성일:** 2025.12.05  
**프로젝트 리더:** Nexus JoPro  
**핵심 비전:** "월 15만 원의 문자 비용을 0원으로 만드는, 영업인을 위한 가장 쉬운 AI 비서"

---

## 1. 사업 정의 (Concept Definition)

### 1.1. 문제 인식 (Pain Point)

- **비용 과다:** 영업 사원들은 고객 관리(문자)를 위해 월 10~30만 원(뿌리오 등)을 지출함.
- **자원 낭비:** 정작 본인 휴대폰의 '문자/데이터 무제한 요금제'는 쓰지도 못하고 버려짐.
- **사용성 최악:** 기존 CRM은 기능이 너무 복잡하여, 연세 있는 현장 소장님이나 보험 설계사가 쓰기 어려움.

### 1.2. 해결책 (Solution)

- **하이브리드 엔진:** PC 웹의 편리함(대량 입력) + 모바일 앱의 경제성(무료 발송) 결합.
- **초간단 UX:** "복잡한 설정은 AI가, 사장님은 '전송' 버튼 하나만."
- **업종별 최적화:** 보험, 자동차, 부동산 등 각 분야에 맞는 자동화 시나리오 탑재.

---

## 2. 핵심 기능 명세 (Key Features)

### 2.1. 기본기: 문자 발송 및 관리 (Core)

- **PC-Mobile 동기화:** 웹 대시보드에서 입력하면 내 폰(안드로이드)이 자동으로 발송.
- **안전 발송 시스템 (Safety Queue):**
  - **스로틀링:** 15초에 1건씩 천천히 발송 (안드로이드 시스템 차단 회피).
  - **일일 한도 제어:** '안전 모드(199건)' 및 '최대 모드(490건)' 자동 카운팅 및 차단.
- **스마트 콜백:** 통화 종료 후 상대방(신규/기존)에 따라 명함 및 안내 문자 자동 발송.

### 2.2. 필살기: AI 디자인 스튜디오 (Killer Feature)

- **AI 배경 생성:** DeepSeek/DALL-E 연동. 상황(골프, 명절, 생일)에 맞는 배경 생성.
- **하이브리드 합성:** 생성된 이미지 위에 **'코드(Code)'**로 텍스트 합성 (수정 용이, 고화질).
- **개인화 치환 (%Name%):**
  - 내용: "안녕하세요 **%고객명%**님!" → 수신자 이름으로 자동 변환.
  - 효과: 스팸 필터 회피 + 고객 감동 극대화.

### 2.3. 영업 비밀병기: DB 자동 수집기

- **미끼 전략:** "커피 쿠폰 보내드릴게요" 버튼 클릭.
- **랜딩 페이지:** 고객이 직접 본인 정보(생일, 주소, 관심사) 입력.
- **자동화:** 입력된 정보가 즉시 내 앱 고객 명단(생일, 기념일)에 자동 등록.

## 3. 사용자 경험 (UX/UI) 전략 - "효도폰 스타일"

### 3.1. 철학: "비서의 브리핑"

- 복잡한 메뉴판(대시보드) 제거.
- 앱 실행 시 [오늘의 할 일] 카드만 노출.
- "사장님, 오늘 김철수님 생일입니다. 축하 문자 보낼까요?" → [보내기] (끝)

### 3.2. 업종별 모드 (Industry Mode)

가입 시 업종 선택에 따라 '자동화 템플릿' 변경.

| 모드 | 자동화 시나리오 (AI 비서) |
|------|-------------------------|
| 보험 전용 | 상령일(보험료 인상) D-7 알림, 보험금 청구 서류 원터치 발송 |
| 자동차 전용 | 출고 1주년/3주년 점검 알림, 엔진오일 교체 주기 알림 |
| 부동산 전용 | 전세 만기 3개월 전 알림, 임장(집 구경) 후 매물 요약장 자동 발송 |
| 현장 관리 | (건설) 우천 시 작업 취소 공지 원터치 발송, 일일 작업보고 자동 취합 |

## 4. 수익 모델 (Business Model)

| 구분 | 가격 정책 | 예상 수익 (가입자 1,000명 기준) |
|------|----------|-------------------------------|
| 월 구독료 | 월 9,900원 (기본 기능) | 월 990만 원 (고정) |
| AI 디자인 | 건당 500원~1,000원 (3회 무료 후 유료) | 월 100~200만 원 (알짜 마진 90%) |
| 커머스 | 쿠폰 판매 마진 (스벅, 비타500 등) | 월 300만 원 (중개 수수료) |
| **예상 월수익** | **약 1,400만 원** (서버비 제외 순수익) | (서버비는 거의 0원에 수렴) |

## 5. 기술 스택 (Tech Stack) - 최적화

- **Mobile:** React Native (CLI) - 안드로이드 문자/통화 제어 최적화.
- **Web:** Next.js - PC 대시보드 및 모바일 랜딩 페이지.
- **Backend & DB:** Supabase - 실시간 DB, 인증, 스토리지 (서버 구축 불필요).
- **AI Engine:**
  - 텍스트: DeepSeek API (가성비 최강).
  - 이미지: DALL-E 3 or Flux API.
- **Infra:** 사용자 휴대폰(Traffic 분산) + Vercel(웹 호스팅).

## 6. 단계별 실행 로드맵 (Roadmap)

### Phase 1: MVP (생존과 검증) - D+1개월

**목표:** 선생님 본인과 지인(건설 소장님)들이 실사용.

- **개발:** PC 발송 + 일일 490건 제한 + 기본 콜백.
- **배포:** APK 파일 직접 전달 (문자 링크).

### Phase 2: 상용화 (비즈니스 시작) - D+3개월

**목표:** 원스토어 입점 및 유료 결제 연동.

**개발:**
- AI 디자인 스튜디오 (배경 생성 + 텍스트 합성).
- 고객 정보 수집 페이지 (커피 쿠폰 연동 수동).
- 업종별 템플릿(보험/자동차) 탑재.

### Phase 3: 확장 (플랫폼 완성) - D+6개월

**목표:** 가입자 1,000명 돌파, 커머스 연동.

**개발:**
- 기프티콘 B2B API 연동 (앱 내 상점 오픈).
- RCS(채팅+) 메시지 규격 지원.
- 스팸 번호 공유(집단지성) 기능.

## 7. 리스크 관리 (Risk Management)

- **통신사 제재:**
  - 대응: '안전 모드(일 199건)' 기본 설정 및 앱 내 경고 팝업 강화.
- **아이폰 유저:**
  - 대응: "안드로이드 전용 서비스"임을 명시하고, 영업용 세컨폰(갤럭시 공기계) 활용 유도.
- **초기 마케팅:**
  - 대응: "문자비 월 15만 원 절약" 키워드로 블로그/유튜브 공략, 보험사 지점 단위 영업.
