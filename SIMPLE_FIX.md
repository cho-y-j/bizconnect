# 간단한 해결 방법

## 문제
아까는 잘 되었는데 지금 안 됩니다.

## 해결 방법 (순서대로)

### 1단계: 개발 서버 재시작 (가장 중요!)

```powershell
# 1. 서버 중지 (Ctrl+C)
# 2. 다시 시작
cd web
npm run dev
```

### 2단계: 브라우저 새로고침

- Ctrl+Shift+R (강력 새로고침)
- 또는 F5

### 3단계: 여전히 안 되면 시크릿 모드

- Ctrl+Shift+N (Chrome 시크릿 모드)
- http://localhost:3000 접속
- 로그인 시도

---

## 가장 가능성 높은 원인

**개발 서버를 재시작하지 않아서** 환경 변수나 코드 변경이 반영되지 않았을 수 있습니다.

**서버를 재시작**하면 대부분 해결됩니다!
