# 간단한 해결

## 문제
아까는 잘 되었는데 지금 안 됩니다.

## 해결 방법

### 1단계: 개발 서버 완전 재시작

```powershell
# 1. 서버 중지 (Ctrl+C)
# 2. 포트 확인 (3000번 포트 사용 중인지)
netstat -ano | findstr :3000

# 3. 필요시 프로세스 종료
# 4. 다시 시작
cd web
npm run dev
```

### 2단계: 브라우저 완전 새로고침

- Ctrl+Shift+R (강력 새로고침)
- 또는 브라우저 완전 종료 후 다시 시작
