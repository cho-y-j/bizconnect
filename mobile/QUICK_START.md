# 빠른 시작 가이드 (요약)

## 1단계: Android Studio 열기
```
1. Android Studio 실행
2. File → Open
3. C:\cho\call\BizConnect\mobile 폴더 선택
```

## 2단계: Gradle 동기화 대기
```
- 자동으로 시작됨
- 하단 진행 상황 표시 확인
- 완료될 때까지 대기 (5-10분)
```

## 3단계: Metro 번들러 시작 (별도 터미널)
```powershell
cd C:\cho\call\BizConnect\mobile
npm start
```

## 4단계: 디바이스 연결
```
- Android 기기: USB 디버깅 활성화 후 USB 연결
- 또는 에뮬레이터 실행
```

## 5단계: 앱 실행
```
1. Android Studio 상단에서 디바이스 선택
2. ▶️ 실행 버튼 클릭
3. 빌드 및 설치 대기
```

## 문제 발생 시
```
1. File → Invalidate Caches / Restart
2. 재시작 후 다시 시도
```
