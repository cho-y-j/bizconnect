## 다음 해야 할 것 (우선순위)

### 1. 데이터베이스 마이그레이션 (필수!) ⭐⭐⭐

**그룹과 태그 테이블 추가**

파일: `supabase/add-groups-tags.sql`

**해야 할 일:**
1. Supabase Dashboard > SQL Editor
2. `supabase/add-groups-tags.sql` 복사
3. 실행

**이것을 안 하면:**
- 그룹 기능 작동 안 함
- 태그 기능 작동 안 함
- 고객 추가 시 그룹 선택 불가

### 2. 문자 발송 UI 구현 ⭐⭐⭐

**문자 발송 페이지** (`/dashboard/send`)

**구현할 기능:**

- 그룹별 선택

- 미리보기

**중요:**
- 실제 문자 발송은 **모바일 앱**에서 함
